== Description ==

Simple referrer spam blocker that redirects spam links back to their own site.  A few known spammers are loaded on plugin activation but you can add or remove any as needed.


== Installation ==


1. Upload zip file through plugins page or unzipped folder via FTP
2. Activate the plugin through the 'Plugins' menu in WordPress
